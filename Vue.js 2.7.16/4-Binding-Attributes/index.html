Binding to Attributes:

We can not bond any attribute like this, as vue does not support attribute binding using string interpolation syntax. But again if we want to bind the data for any attribute we can make use of existing directive called v-bind.

Example:

{{ sayHello() }} You can visit me here Tanish Raj (Incorrect, using string interpolation)

{{ sayHello() }} You can visit me here Tanish Raj (Correct, using v-bind)

{{ sayHello() }} You can visit me here Tanish Raj (Correct, using : symbol)
